# VertexFix 🔧

A fully responsive repair-service landing page built entirely with **HTML, CSS, and Bootstrap** — no JavaScript.

🔗 **Live Demo:** https://aparajith-2105.github.io/vertexfix/

## About

VertexFix is a landing page for a fictional repair-service business, designed and built as a front-end practice project focused on responsive web design. It's my second project, following my first (SubSnap, an AI-driven project), built with the goal of strengthening my fundamentals in layout, responsiveness, and styling without relying on any frameworks beyond Bootstrap.

## Features

- Fully responsive layout across mobile, tablet, and desktop using Bootstrap's grid system
- Collapsible responsive navbar with a dropdown menu
- Custom CSS hover animations and transitions
- Card-based "Benefits" section with equal-height layouts across breakpoints
- Clean black-and-white brand aesthetic with a full-width hero section

## Built With

- HTML5
- CSS3
- Bootstrap

## What I Learned

Building this taught me a lot about why responsive layouts break, not just how to fix them, including:

- How Bootstrap's grid system requires `.col-*` elements to be direct children of `.row` to work correctly
- The difference between float, inline-block, and flex/grid layout, and when each one causes inconsistent sizing across screen widths
- Debugging real responsiveness issues, such as column widths not summing to 12, headers growing too tall due to stacked block elements, and unstable card heights caused by shrink-to-fit sizing
